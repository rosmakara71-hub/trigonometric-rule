📖 មេរៀនសង្ខេប និងធនាគារលំហាត់៖ ភាពគូសេសនៃអនុគមន៍
💡 សេចក្ដីសង្ខេបទ្រឹស្ដីគ្រឹះ៖

អនុគមន៍ f ដែលមានដែនកំណត់ D ជាសំណុំឆ្លុះធៀបនឹងគល់ 0 (កាលណា x ∈ D នាំឱ្យ -x ∈ D)៖

  • ជា អនុគមន៍គូ (Even Function) កាលណា f(-x) = f(x)។ ក្រាបឆ្លុះធៀបនឹងអ័ក្ស (Oy)។
  • ជា អនុគមន៍សេស (Odd Function) កាលណា f(-x) = -f(x)។ ក្រាបឆ្លុះធៀបនឹងគល់ O(0,0)។
១. កម្រិតមូលដ្ឋាន (Concept & Polynomials) កម្រិត ១
លំហាត់ ១.១៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = 2x4 - 5x2 + 1 លើ D = ℝ។
📝 ដំណោះស្រាយលម្អិត៖
- ដែនកំណត់ D = ℝ (ឆ្លុះ)
- គណនា៖ f(-x) = 2(-x)4 - 5(-x)2 + 1 = 2x4 - 5x2 + 1 = f(x)
👉 ដូចនេះ f ជា អនុគមន៍គូ។
លំហាត់ ១.២៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = x3 - 4x លើ D = ℝ។
📝 ដំណោះស្រាយលម្អិត៖
- ដែនកំណត់ D = ℝ (ឆ្លុះ)
- គណនា៖ f(-x) = (-x)3 - 4(-x) = -x3 + 4x = -(x3 - 4x) = -f(x)
👉 ដូចនេះ f ជា អនុគមន៍សេស។
២. កម្រិតមធ្យម (Rational & Absolute Value) កម្រិត ២
លំហាត់ ២.១៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = |x| x3 + x លើដែនកំណត់របស់វា។
📝 ដំណោះស្រាយលម្អិត៖
- លក្ខខណ្ឌភាគបែង៖ x3 + x ≠ 0 ⇒ x(x2 + 1) ≠ 0 ⇒ x ≠ 0
- នាំឱ្យដែនកំណត់ D = ℝ* (ឆ្លុះ)
- គណនា៖ f(-x) = |-x|(-x)3+(-x) = |x|-(x3+x) = -f(x)
👉 ដូចនេះ f ជា អនុគមន៍សេស។
លំហាត់ ២.២៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = x2 + 2 |x - 3| + |x + 3| លើដែនកំណត់របស់វា។
📝 ដំណោះស្រាយលម្អិត៖
- ភាគបែង |x - 3| + |x + 3| ≠ 0 ជានិច្ច នាំឱ្យ D = ℝ (ឆ្លុះ)
- គណនា៖ f(-x) = (-x)2 + 2 |-x - 3| + |-x + 3| = x2 + 2 |-(x + 3)| + |-(x - 3)| = x2 + 2 |x + 3| + |x - 3| = f(x) (ដោយសារផលបូកមានលក្ខណៈត្រឡប់)
👉 ដូចនេះ f ជា អនុគមន៍គូ។
៣. កម្រិតខ្ពស់ (Irrational & Complex Algebra) កម្រិត ៣
លំហាត់ ៣.១៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = √9 - x2 លើដែនកំណត់របស់វា។
📝 ដំណោះស្រាយលម្អិត៖
- លក្ខខណ្ឌ៖ 9 - x2 ≥ 0 ⇒ x2 ≤ 9 ⇒ x ∈ [-3, 3] នាំឱ្យ D = [-3, 3] (ឆ្លុះ)
- គណនា៖ f(-x) = √9 - (-x)2 = √9 - x2 = f(x)
👉 ដូចនេះ f ជា អនុគមន៍គូ។
លំហាត់ ៣.២៖ សិក្សាភាពគូសេសនៃអនុគមន៍ f(x) = √x2 + 1 - x លើ D = ℝ។
📝 ដំណោះស្រាយលម្អិត៖
- ដែនកំណត់ D = ℝ (ឆ្លុះ) ព្រោះ x2 + 1 > 0 ជានិច្ច
- គណនា៖ f(-x) = √(-x)2 + 1 - (-x) = √x2 + 1 + x
- គុណនឹងកន្សោមឆ្លាស់ដើម្បីពិនិត្យទម្រង់៖
f(-x) = (√x2 + 1 + x)(√x2 + 1 - x) √x2 + 1 - x = (x2 + 1) - x2 √x2 + 1 - x = 1 f(x)
- ម្យ៉ាងទៀត ពិនិត្យផលគុណ៖ f(x) × f(-x) = 1
👉 ដោយសារ f(-x) ≠ f(x) និង f(-x) ≠ -f(x) ដូចនេះអនុគមន៍នេះ មិនមែនគូ និងមិនមែនសេសឡើយ។
