- ដូចនេះ $f$ ជាអនុគមន៍សេស (ភាគយកគូ ភាគបែងសេស ផលចែកចេញសេស)។
សិក្សាភាពគូសេសនៃអនុគមន៍ $f(x) = \frac{x^2+2}{x^3-4x}$ លើដែនកំណត់ $D = \mathbb{R} \setminus \{-2, 0, 2\}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ភាគបែង $x(x^2-4) \neq 0 \Rightarrow x \neq 0, \pm 2$ នាំឱ្យ $D$ ឆ្លុះធៀបនឹង $0$

- ដូចនេះ $f$ ជាអនុគមន៍សេស។

💡 ដំណោះស្រាយលម្អិត៖
- $D = \mathbb{R}$ (ឆ្លុះ)
- គណនា $f(-x) = \frac{(-x-1)^2}{(-x)^2+1} = \frac{(x+1)^2}{x^2+1}$
- ឃើញថា $f(-x) \neq f(x)$ និង $f(-x) \neq -f(x)$
- ដូចនេះ $f$ មិនមែនជាអនុគមន៍គូ ហើយក៏មិនមែនជាអនុគមន៍សេសឡើយ។
សិក្សាភាពគូសេសនៃអនុគមន៍ $f(x) = \frac{|x+3| + |x-3|}{x^2}$ លើដែនកំណត់ $D = \mathbb{R}^*$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- $D = \mathbb{R}^*$ (ឆ្លុះ)
- គណនា $f(-x) = \frac{|-x+3| + |-x-3|}{(-x)^2} = \frac{|-(x-3)| + |-(x+3)|}{x^2} = \frac{|x-3| + |x+3|}{x^2} = f(x)$
- ដូចនេះ $f$ ជាអនុគមន៍គូ។
សិក្សាភាពគូសេសនៃអនុគមន៍ $f(x) = \frac{x^3 - 4x}{|x-2| - |x+2|}$ លើដែនកំណត់របស់វា។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- លក្ខខណ្ឌភាគបែង៖ $|x-2| - |x+2| \neq 0 \Rightarrow |x-2| \neq |x+2| \Rightarrow x \neq 0$ នាំឱ្យ $D = \mathbb{R}^*$ (ឆ្លុះ)
- គណនា $f(-x) = \frac{(-x)^3 - 4(-x)}{|-x-2| - |-x+2|} = \frac{-x^3 + 4x}{|-(x+2)| - |-(x-2)|} = \frac{-(x^3 - 4x)}{|x+2| - |x-2|} = \frac{-(x^3 - 4x)}{-(|x-2| - |x+2|)} = \frac{x^3 - 4x}{|x-2| - |x+2|} = f(x)$
- ដោយសារភាគយកសេស និងភាគបែងសេស ផលចែកចេញមកជាអនុគមន៍គូ។
