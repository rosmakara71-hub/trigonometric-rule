📝 វិញ្ញាសាប្រឡងតេស្តសមត្ថភាព

មេរៀន៖ វ៉ិចទ័រទីតាំង និងការចែកអង្កត់ (ពិន្ទុពេញ ៥០)
រៀបរៀងដោយ៖ លោកគ្រូ រស់ មករា
សំណួរទី ១៖ គេមានវ៉ិចទ័រទីតាំង a = (3, -1) និង b = (7, 5)។ ចូររកវ៉ិចទ័រទីតាំងនៃចំណុចកណ្តាល I នៃអង្កត់ AB។ (៥ ពិន្ទុ)
សំណួរទី ២៖ ចំណុច C ចែកអង្កត់ AB ខាងក្នុងតាមផលធៀប m : n = 2 : 1។ បើ a និង b ជាវ៉ិចទ័រទីតាំងនៃ A និង B តើវ៉ិចទ័រទីតាំង c ស្មើនឹងប៉ុន្មាន? (៥ ពិន្ទុ)
សំណួរទី ៣៖ គេឲ្យចំណុច A(1, 2) និង B(4, 5)។ វ៉ិចទ័រទីតាំង d នៃចំណុច D ដែលចែកអង្កត់ AB ខាងក្រៅតាមផលធៀប 2 : 1 គឺ៖ (៥ ពិន្ទុ)
សំណួរទី ៤៖ បើចំណុច P(5, y) ជាចំណុចកណ្តាលនៃអង្កត់តភ្ជាប់ពី A(x, 4) ទៅ B(3, 8) ចូររកតម្លៃ x និង y។ (៥ ពិន្ទុ)
សំណួរទី ៥៖ បើ OP = p ជាវ៉ិចទ័រទីតាំងនៃចំណុច P ធៀបនឹងគល់ O ថេរមួយ។ តើនិយមន័យនេះត្រឹមត្រូវដែរឬទេ? (៥ ពិន្ទុ)
សំណួរទី ៦៖ គេឲ្យ a = (2, 4) និង b = (6, 8)។ រកកូអរដោនេនៃចំណុច C ដែលចែកអង្កត់ AB ខាងក្នុងតាមផលធៀប 1 : 1។ (៥ ពិន្ទុ)
សំណួរទី ៧៖ តាមរូបមន្តចែកអង្កត់ខាងក្រៅ d = (mb - na) / (m - n)។ តើលក្ខខណ្ឌចាំបាច់នៃផលធៀប m និង n គឺអ្វី? (៥ ពិន្ទុ)
សំណួរទី ៨៖ គេឲ្យចំណុច A(2, -1, 3) និង B(5, 2, -3) ក្នុងលំហ។ រកកូអរដោនេចំណុច D ចែកខាងក្រៅ AB តាមផលធៀប 2 : 1។ (៥ ពិន្ទុ)
សំណួរទី ៩៖ ក្នុងត្រីកោណ ABC បើ G ជាទីប្រជុំទម្ងន់ តើវ៉ិចទ័រទីតាំង g កំណត់ដោយរូបមន្តណា? (៥ ពិន្ទុ)
សំណួរទី ១០៖ គេមានចំណុច A(1, -3) និង B(5, 9)។ ចូរគណនាវ៉ិចទ័រទីតាំង c នៃចំណុច C ដែលចែកអង្កត់ AB ខាងក្នុងតាមផលធៀប 1 : 2។ (៥ ពិន្ទុ)
