бһңбһ·бһүбҹ’бһүбһ¶бһҹбһ¶бһңбһ¶бһҹбҹӢбһҹбҹ’бһ‘бһ„бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһ·бһҹбҹ’бһҹ ( )

F

рҹ‘Ү бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбҹҶбһ“бһҪбһҡбһ“бһёбһҳбһҪбһҷбҹ—!

бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $\cos(60^\circ - 45^\circ)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸ $\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos(60^\circ - 45^\circ) = \cos60^\circ \cos45^\circ + \sin60^\circ \sin45^\circ$
- бһҮбҹҶбһ“бһҪбһҹбһҸбһҳбҹ’бһӣбҹғбҹ– $\left(\frac{1}{2}\right)\left(\frac{\sqrt{2}}{2}\right) + \left(\frac{\sqrt{3}}{2}\right)\left(\frac{\sqrt{2}}{2}\right) = \frac{\sqrt{2}}{4} + \frac{\sqrt{6}}{4} = \frac{\sqrt{6} + \sqrt{2}}{4}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $\cos(45^\circ + 30^\circ)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸ $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos(45^\circ + 30^\circ) = \cos45^\circ \cos30^\circ - \sin45^\circ \sin30^\circ$
- бһҮбҹҶбһ“бһҪбһҹбһҸбһҳбҹ’бһӣбҹғбҹ– $\left(\frac{\sqrt{2}}{2}\right)\left(\frac{\sqrt{3}}{2}\right) - \left(\frac{\sqrt{2}}{2}\right)\left(\frac{1}{2}\right) = \frac{\sqrt{6}}{4} - \frac{\sqrt{2}}{4} = \frac{\sqrt{6} - \sqrt{2}}{4}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $A = \cos 75^\circ \cos 15^\circ + \sin 75^\circ \sin 15^\circ$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ–бҹ’бһҡбһӣбһ¶бһҸ $\cos\alpha \cos\beta + \sin\alpha \sin\beta = \cos(\alpha - \beta)$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $A = \cos(75^\circ - 15^\circ) = \cos 60^\circ$
- бһҠбһјбһ…бһ“бҹҒбҹҮ $A = \frac{\sqrt{3}}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $B = \cos\frac{5\pi}{12} \cos\frac{\pi}{12} - \sin\frac{5\pi}{12} \sin\frac{\pi}{12}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ–бҹ’бһҡбһӣбһ¶бһҸ $\cos\alpha \cos\beta - \sin\alpha \sin\beta = \cos(\alpha + \beta)$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $B = \cos\left(\frac{5\pi}{12} + \frac{\pi}{12}\right) = \cos\left(\frac{6\pi}{12}\right) = \cos\left(\frac{\pi}{2}\right)$
- бһҠбһјбһ…бһ“бҹҒбҹҮ $B = 0$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $y = \cos(x - \frac{\pi}{6}) - \cos(x + \frac{\pi}{6})$ бһұбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбһ¶бһҳбһүбҹ’бһүбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ–бҹ’бһҡбһӣбһ¶бһҸбһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһ”бһјбһҖ бһ“бһ·бһ„бһ•бһӣбһҠбһҖбҹ–
- $\cos(x - \frac{\pi}{6}) = \cos x \cos\frac{\pi}{6} + \sin x \sin\frac{\pi}{6}$
- $\cos(x + \frac{\pi}{6}) = \cos x \cos\frac{\pi}{6} - \sin x \sin\frac{\pi}{6}$
- бһ’бҹ’бһңбһҫбһ•бһӣбһҠбһҖбҹ– $y = 2 \sin x \sin\frac{\pi}{6} = 2 \sin x \left(\frac{1}{2}\right) = \sin x$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $C = \cos 105^\circ + \cos 15^\circ$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбһ№бһ„бһҗбһ¶ $\cos 105^\circ = \cos(60^\circ + 45^\circ) = \frac{\sqrt{2}-\sqrt{6}}{4}$
бһ“бһ·бһ„ $\cos 15^\circ = \cos(45^\circ - 30^\circ) = \frac{\sqrt{6}+\sqrt{2}}{4}$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $C = \frac{\sqrt{2}-\sqrt{6}}{4} + \frac{\sqrt{6}+\sqrt{2}}{4} = \frac{2\sqrt{2}}{4} = \frac{\sqrt{2}}{2}$бҹ”
бһҸбһҫбһҖбһ“бҹ’бһҹбҹ„бһҳ $\cos\alpha \cos\beta - \sin\alpha \sin\beta$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҳбһҪбһҷбһҺбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ“бҹҒбҹҮбһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһ“бҹғбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһ”бһјбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһӮбһҳбһ“бҹҚбһҖбһјбһҹбҹҠбһёбһ“бһ»бһҹбҹ– $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\cos\left(\frac{\pi}{3} + \frac{\pi}{4}\right)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
$\cos\left(\frac{\pi}{3} + \frac{\pi}{4}\right) = \cos\frac{\pi}{3}\cos\frac{\pi}{4} - \sin\frac{\pi}{3}\sin\frac{\pi}{4}$
$= \left(\frac{1}{2}\right)\left(\frac{\sqrt{2}}{2}\right) - \left(\frac{\sqrt{3}}{2}\right)\left(\frac{\sqrt{2}}{2}\right) = \frac{\sqrt{2} - \sqrt{6}}{4}$бҹ”
бһҹбһ¶бһҳбһүбҹ’бһүбһҖбһҳбҹ’бһҳбһҖбһ“бҹ’бһҹбҹ„бһҳ $P = \cos(2\theta) \cos\theta + \sin(2\theta) \sin\theta$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһҠбһҖ $\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta$
бһҷбһҖ $\alpha = 2\theta$ бһ“бһ·бһ„ $\beta = \theta$ бһ“бһ¶бҹҶбһұбҹ’бһҷ $P = \cos(2\theta - \theta) = \cos\theta$бҹ”
бһӮбҹҒбһұбҹ’бһҷ $\sin\alpha = \frac{3}{5}$ бһҠбҹӮбһӣ $\frac{\pi}{2} < \alpha < \pi$ бһ“бһ·бһ„ $\cos\beta = \frac{5}{13}$ бһҠбҹӮбһӣ $0 < \beta < \frac{\pi}{2}$бҹ” бһӮбһҺбһ“бһ¶ $\cos(\alpha - \beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҡбһҖ $\cos\alpha$бҹ– бһҠбҹ„бһҷ $\alpha$ бһ“бҹ…бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹў бһ“бҹ„бҹҮ $\cos\alpha < 0 \implies \cos\alpha = -\sqrt{1 - \sin^2\alpha} = -\frac{4}{5}$
- бһҡбһҖ $\sin\beta$бҹ– бһҠбҹ„бһҷ $\beta$ бһ“бҹ…бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЎ бһ“бҹ„бҹҮ $\sin\beta > 0 \implies \sin\beta = \sqrt{1 - \cos^2\beta} = \frac{12}{13}$
- бһӮбһҺбһ“бһ¶ $\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta = \left(-\frac{4}{5}\right)\left(\frac{5}{13}\right) + \left(\frac{3}{5}\right)\left(\frac{12}{13}\right) = \frac{-20 + 36}{65} = \frac{16}{65}$бҹ”
бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ‘бһёбҹЎбҹ  ( $\sin\alpha = \frac{3}{5}$ бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹў бһ“бһ·бһ„ $\cos\beta = \frac{5}{13}$ бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЎ ) бһ…бһјбһҡбһӮбһҺбһ“бһ¶ $\cos(\alpha + \beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ–бһёбһӣбҹҶбһ бһ¶бһҸбҹӢбһҳбһ»бһ“бһҷбһҫбһ„бһ”бһ¶бһ“ $\cos\alpha = -\frac{4}{5}$ бһ“бһ·бһ„ $\sin\beta = \frac{12}{13}$
- бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$
- бһҮбҹҶбһ“бһҪбһҹбһҸбһҳбҹ’бһӣбҹғбҹ– $\left(-\frac{4}{5}\right)\left(\frac{5}{13}\right) - \left(\frac{3}{5}\right)\left(\frac{12}{13}\right) = \frac{-20 - 36}{65} = -\frac{56}{65}$бҹ”
бһӮбҹҒбһұбҹ’бһҷ $\cos\alpha = -\frac{12}{13}$ бһ“бһ·бһ„ $\sin\beta = -\frac{4}{5}$ бһҠбҹ„бһҷбһҳбһ»бҹҶ $\alpha, \beta$ бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЈ ($180^\circ < \alpha, \beta < 270^\circ$)бҹ” бһӮбһҺбһ“бһ¶ $\cos(\alpha - \beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЈ бһўбһ“бһ»бһӮбһҳбһ“бҹҚ $\sin$ бһ“бһ·бһ„ $\cos$ бһҹбһ»бһ‘бҹ’бһ’бһҸбҹӮбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\sin\alpha = -\sqrt{1 - (-\frac{12}{13})^2} = -\frac{5}{13}$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos\beta = -\sqrt{1 - (-\frac{4}{5})^2} = -\frac{3}{5}$
- бһӮбһҺбһ“бһ¶ $\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta = (-\frac{12}{13})(-\frac{3}{5}) + (-\frac{5}{13})(-\frac{4}{5}) = \frac{36 + 20}{65} = \frac{56}{65}$бҹ”
бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ‘бһёбҹЎбҹў бһ…бһјбһҡбһӮбһҺбһ“бһ¶ ЩӮЩҠЩ…Ш© $\cos(\alpha + \beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһҠбһҠбҹӮбһӣбҹ– $\cos\alpha = -\frac{12}{13}, \sin\alpha = -\frac{5}{13}, \cos\beta = -\frac{3}{5}, \sin\beta = -\frac{4}{5}$
- $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$
- $= (-\frac{12}{13})(-\frac{3}{5}) - (-\frac{5}{13})(-\frac{4}{5}) = \frac{36 - 20}{65} = \frac{16}{65}$бҹ”
бһӮбҹҒбһұбҹ’бһҷ $\tan\alpha = \frac{3}{4}$ бһ“бһ·бһ„ $\tan\beta = \frac{5}{12}$ бһҠбҹӮбһӣ $\alpha, \beta$ бһҮбһ¶бһҳбһ»бҹҶбһҹбҹ’бһҡбһҪбһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҠбҹ’бһҡбһ„бҹӢбһ‘бһёбҹЎбҹ” бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғ $\cos(\alpha + \beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҸбһ¶бһҳбһҸбҹҒбһҸбҹ’бһҡбһ¶бһҮбҹ’бһҡбһ»бһ„бһҖбҹӮбһ„ бһӮбҹҒбһ‘бһ¶бһүбһ”бһ¶бһ“бҹ– $\sin\alpha = \frac{3}{5}, \cos\alpha = \frac{4}{5}$ бһ“бһ·бһ„ $\sin\beta = \frac{5}{13}, \cos\beta = \frac{12}{13}$
- бһӮбһҺбһ“бһ¶ $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta = \left(\frac{4}{5}\right)\left(\frac{12}{13}\right) - \left(\frac{3}{5}\right)\left(\frac{5}{13}\right)$
- $= \frac{48 - 15}{65} = \frac{33}{65}$бҹ”
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ $\cos x = \frac{1}{3}$ бһ…бһјбһҡбһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $\cos(x - \pi)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos(x - \pi) = \cos x \cos\pi + \sin x \sin\pi$
- бһҠбҹ„бһҷбһҠбһ№бһ„бһҗбһ¶ $\cos\pi = -1$ бһ“бһ·бһ„ $\sin\pi = 0$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos(x - \pi) = \cos x (-1) + 0 = -\cos x = -\frac{1}{3}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғ $\cos 15^\circ$ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҳбһ“бҹ’бһҸбһ•бһӣбһҠбһҖбҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
$\cos 15^\circ = \cos(45^\circ - 30^\circ) = \cos 45^\circ \cos 30^\circ + \sin 45^\circ \sin 30^\circ$
$= \left(\frac{\sqrt{2}}{2}\right)\left(\frac{\sqrt{3}}{2}\right) + \left(\frac{\sqrt{2}}{2}\right)\left(\frac{1}{2}\right) = \frac{\sqrt{6} + \sqrt{2}}{4}$бҹ”
бһ”бһҫ $\cos(A + B) = 0$ бһ“бһ·бһ„ $\sin A \sin B = \frac{1}{2}$ бһҸбһҫбһҸбһҳбҹ’бһӣбҹғ $\cos A \cos B$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos(A + B) = \cos A \cos B - \sin A \sin B$
- бһҮбҹҶбһ“бһҪбһҹбһ”бһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһјбһӣбҹ– $0 = \cos A \cos B - \frac{1}{2}$
- бһҠбһјбһ…бһ“бҹҒбҹҮ $\cos A \cos B = \frac{1}{2}$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғбһҖбһ“бҹ’бһҹбҹ„бһҳ $\cos 80^\circ \cos 20^\circ + \sin 80^\circ \sin 20^\circ$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҖбһ“бҹ’бһҹбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ $\cos(\alpha - \beta)$
- бһ“бһ¶бҹҶбһұбҹ’бһҷбһҹбҹ’бһҳбһҫбһ“бһ№бһ„ $\cos(80^\circ - 20^\circ) = \cos 60^\circ = \frac{1}{2}$бҹ”
бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $M = \frac{\cos(\alpha - \beta) - \cos(\alpha + \beta)}{2}$бҹ” бһҸбһҫбһңбһ¶бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҖбһ“бҹ’бһҹбҹ„бһҳбһҺбһ¶бһҒбҹ’бһӣбҹҮ? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- $\cos(\alpha - \beta) = \cos\alpha \cos\beta + \sin\alpha \sin\beta$
- $\cos(\alpha + \beta) = \cos\alpha \cos\beta - \sin\alpha \sin\beta$
- бһ’бҹ’бһңбһҫбһ•бһӣбһҠбһҖбҹ– $\cos(\alpha - \beta) - \cos(\alpha + \beta) = 2 \sin\alpha \sin\beta$
- бһ…бҹӮбһҖбһ“бһ№бһ„ бҹў бһ“бһ¶бҹҶбһұбҹ’бһҷ $M = \sin\alpha \sin\beta$бҹ”
бһҖбҹ’бһ“бһ»бһ„бһҸбҹ’бһҡбһёбһҖбҹ„бһҺ $ABC$ бһҳбһҪбһҷ бһҖбһ“бҹ’бһҹбҹ„бһҳ $\cos A \cos B - \sin A \sin B$ бһҹбҹ’бһҳбһҫбһ“бһ№бһ„бһҸбһҳбҹ’бһӣбҹғбһҳбһҪбһҷбһҺбһ¶? (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
- бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸбҹ– $\cos A \cos B - \sin A \sin B = \cos(A + B)$
- бһҖбҹ’бһ“бһ»бһ„бһҸбҹ’бһҡбһёбһҖбҹ„бһҺ $ABC$ бһҷбһҫбһ„бһҳбһ¶бһ“ $A + B + C = \pi \implies A + B = \pi - C$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos(A + B) = \cos(\pi - C) = -\cos C$бҹ”
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ $\alpha + \beta = \frac{\pi}{4}$ бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ“бҹғ $(1 + \tan\alpha)(1 + \tan\beta)$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҸбһ¶бһҳбһҡбһјбһ”бһҳбһ“бҹ’бһҸ $\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta} = \tan\frac{\pi}{4} = 1$
$\implies \tan\alpha + \tan\beta = 1 - \tan\alpha\tan\beta \implies \tan\alpha + \tan\beta + \tan\alpha\tan\beta = 1$
- бһ–бҹ’бһҡбһӣбһ¶бһҸбһҖбһ“бҹ’бһҹбҹ„бһҳбһҹбһҪбһҡбҹ– $(1 + \tan\alpha)(1 + \tan\beta) = 1 + \tan\alpha + \tan\beta + \tan\alpha\tan\beta = 1 + 1 = 2$бҹ”
бһҹбһ¶бһҳбһүбҹ’бһүбһҖбһҳбҹ’бһҳбһҖбһ“бҹ’бһҹбҹ„бһҳ $\frac{\cos(\alpha - \beta)}{\cos(\alpha + \beta)}$ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣ $\tan\alpha \tan\beta = \frac{1}{3}$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ…бҹӮбһҖбһ—бһ¶бһӮбһҷбһҖбһ“бһ·бһ„бһ—бһ¶бһӮбһ”бҹӮбһ„бһ“бһ№бһ„ $\cos\alpha \cos\beta$ бһӮбҹҒбһ”бһ¶бһ“бҹ–
$\frac{\cos(\alpha-\beta)}{\cos(\alpha+\beta)} = \frac{1 + \tan\alpha\tan\beta}{1 - \tan\alpha\tan\beta}$
- бһҮбҹҶбһ“бһҪбһҹбһҸбһҳбҹ’бһӣбҹғ $\tan\alpha\tan\beta = \frac{1}{3}$ бһ…бһјбһӣбҹ– $\frac{1 + 1/3}{1 - 1/3} = \frac{4/3}{2/3} = 2$бҹ”
бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ•бһӣбһӮбһ»бһҺ $K = \cos 20^\circ \cos 40^\circ \cos 80^\circ$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһӮбһ»бһҺбһ“бһ·бһ„бһ…бҹӮбһҖбһ“бһ№бһ„ $2\sin 20^\circ$бҹ–
$K = \frac{2\sin 20^\circ \cos 20^\circ \cos 40^\circ \cos 80^\circ}{2\sin 20^\circ} = \frac{\sin 40^\circ \cos 40^\circ \cos 80^\circ}{2\sin 20^\circ}$
$= \frac{\sin 80^\circ \cos 80^\circ}{4\sin 20^\circ} = \frac{\sin 160^\circ}{8\sin 20^\circ}$
бһҠбҹ„бһҷбһҹбһ¶бһҡ $\sin 160^\circ = \sin(180^\circ - 20^\circ) = \sin 20^\circ \implies K = \frac{1}{8}$бҹ”
бһҡбһҖбһҸбһҳбҹ’бһӣбҹғбһҠбһ¶бһ…бҹӢбһҒбһ¶бһҸбһ“бҹғ $x$ бһ”бһҫбһҠбһ№бһ„бһҗбһ¶ $\cos(x + 30^\circ) = \sin x$ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ $0^\circ < x < 90^\circ$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһ”бҹҶбһӣбҹӮбһ„ $\sin x = \cos(90^\circ - x)$
- бһ“бһ¶бҹҶбһұбҹ’бһҷбһҹбһҳбһёбһҖбһ¶бһҡбһ‘бҹ…бһҮбһ¶бҹ– $\cos(x + 30^\circ) = \cos(90^\circ - x)$
- бһӮбҹҒбһ”бһ¶бһ“ $x + 30^\circ = 90^\circ - x \implies 2x = 60^\circ \implies x = 30^\circ$бҹ”
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ $A + B + C = \pi$ бһ…бһјбһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ“бҹ’бһҹбҹ„бһҳ $S = \cos(A+B) + \cos C$бҹ” (бҹҘ бһ–бһ·бһ“бҹ’бһ‘бһ»)
рҹ’Ў бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбһҳбҹ’бһўбһ·бһҸбҹ–
бһҠбҹ„бһҷбһҹбһ¶бһҡ $A + B = \pi - C$
- бһ“бһ¶бҹҶбһұбҹ’бһҷ $\cos(A + B) = \cos(\pi - C) = -\cos C$
- бһҠбһјбһ…бһ“бҹҒбҹҮ $S = -\cos C + \cos C = 0$бҹ”
